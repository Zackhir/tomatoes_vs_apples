Tomato vs Apple Classification 🍅🍎

A deep learning image classification project that distinguishes between tomatoes and apples using Convolutional Neural Networks (CNNs).

📌 Project Overview

This project uses a CNN model built with Python and TensorFlow/Keras to classify images into two categories:

Tomatoes 🍅
Apples 🍎

The model was trained on a image dataset and can predict the class of unseen fruit images with high accuracy.

🧠 Technologies Used
Python
TensorFlow / Keras
NumPy
Matplotlib
OpenCV / PIL
Jupyter Notebook
📂 Dataset

The dataset contains images of:

Apples
Tomatoes

The link: https://www.kaggle.com/datasets/samuelcortinhas/apples-or-tomatoes-image-classification

📈 Results
Training Accuracy: 91%
Validation Accuracy: 72%
Test Accuracy: 82%
